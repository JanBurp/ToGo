# ToGo

A simple ToGo list to make a list of places you'd like to go to in the future. (a ToDo list variant)

## Install

Add `OPENROUTE_APIKEY` to your `.env` file

## Testing

  `php artisan test

## Features

- Add places with autocomplete (with openroute)
- Delete places
- Visit places

## Future features
